# Password Strength Analyzer

A web-based Password Strength Analyzer that checks password security based on:

- Password Length
- Uppercase Letters
- Lowercase Letters
- Numbers
- Special Characters
- Common Password Detection

## Features

- Real-time Password Analysis
- Strength Indicator Bar
- Password Score
- Suggestions for Improvement
- Show/Hide Password
- Modern User Interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project.
2. Open index.html in your browser.
